3. Information We Collect

MakaNetu collects different types of information, depending on how you use the application:

3.1. Geographic Location Data (GPS)

‚ö†Ô∏è CRITICAL INFORMATION

Location data collection is essential for the application to function. MakaNetu is a real-time safety system that depends on your precise location to:

  • Send emergency alerts with your exact location to response entities (ERCE/ERFCE)
  • Receive notifications about incidents near your location
  • Display on map georeferenced reports and alerts in your area
  • Report incidents with geographic precision

How We Collect Location:

  • Precision: We use high-precision GPS (bestForNavigation) with 1 meter accuracy
  • Frequency: Automatic updates every 30 seconds or when you move
  • Permissions: We request "always active" or "while in use" location permission
  • Background operation: Location may be collected even when the app is not open (for emergency alerts)

Permissions Requested:

  • iOS: Location When In Use / Location Always
  • Android: ACCESS_FINE_LOCATION / ACCESS_COARSE_LOCATION / ACCESS_BACKGROUND_LOCATION

6. Data Storage and Security

6.1. Where We Store Your Data

  • Database: PostgreSQL with encryption in transit (TLS/SSL)
  • Cache: Redis for temporary location data
  • Media storage: Firebase Storage (Google Cloud)

6.2. Security Measures

We implement the following protections to ensure your data security:

  • TLS/SSL Encryption: All communications between app and server are encrypted
  • JWT Authentication: Security tokens for authenticated sessions
  • Password hashing: Passwords stored with bcrypt (we don't store plain text passwords)
  • Secure WebSocket (WSS): Encrypted real-time communication
  • Input validation: All data is validated to prevent injections and attacks
  • Security logs: Monitoring of suspicious activities
  • Rate limiting: Protection against DDoS and brute force attacks

‚ö†Ô∏è WARNING: While we use security best practices, no system is 100% invulnerable. You are responsible for keeping your account password secure.

6.3. Data Retention

We maintain your data for as long as necessary to provide the service:

  • Account data: While the account is active
  • GPS Location: 24 hours (then anonymized for statistics)
  • Incident reports: Permanently (public safety history)
  • Error logs: 90 days
  • Anonymous user data: 30 days without activity
